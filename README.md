# docsQL

Getting an overview of your Jamstack Markdown files.


## Getting started

```sh
export CONTENT_SOURCES=/path/to/my/project/with/lots/of/markdown/files
npx .
```

## Getting Started

```sh
echo CONTENT_SOURCES=/path/to/my/project/with/lots/of/markdown/files >> .env
npm run dev
```
